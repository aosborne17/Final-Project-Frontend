The Project

The aim of this project was to achieve end to end automation with a python app. The app displays information regarding IT Jobs, and we created a Continuous Integration, Delivery and Deployment pipeline. This was achieved with Jenkins, Docker and AWS instances to ultimately deploy the application on a public webpage.
